# Курс лекций по математическому моделированию

## Описание курса

Данный курс лекций разработан на основе рабочей программы **МДК.02.03 "Математическое моделирование"** и предназначен для студентов, изучающих методы математического моделирования и оптимизации. Курс охватывает как детерминированные задачи с известными параметрами, так и стохастические модели, где решения принимаются в условиях неопределенности и риска.

## Цели и задачи курса

**Цель:** Сформировать у студентов систематические знания о методах математического моделирования и навыки их практического применения для решения задач оптимизации, прогнозирования и принятия решений.

**Задачи:**
- Освоить основные методы детерминированной оптимизации (линейное, нелинейное, динамическое программирование)
- Изучить алгоритмы теории графов и их применение в задачах планирования
- Овладеть методами анализа систем массового обслуживания
- Научиться применять методы прогнозирования временных рядов
- Изучить основы теории игр и теории принятия решений в условиях неопределенности
- Развить навыки построения математических моделей реальных процессов

## Структура курса

Курс состоит из **17 лекций**, разделенных на два основных раздела:

### Раздел 1: Основы моделирования. Детерминированные задачи (Лекции 1-9)

| № | Тема лекции | Ключевые понятия |
|---|-------------|------------------|
| 1 | Введение в математическое моделирование | Типы моделей, этапы моделирования, классификация задач |

| 4 | Динамическое программирование | Принцип оптимальности Беллмана, задача распределения ресурсов |
| 5 | Теория графов | Алгоритм Дейкстры, алгоритм Прима, минимальное остовное дерево |
| 6 | Построение математических моделей | Транспортная задача, задача о назначениях |
| 7 | Практикум по линейному программированию | Двойственность, анализ чувствительности, решение в Excel |
| 8 | Численные методы оптимизации | Градиентный спуск, метод Ньютона, метод золотого сечения |
| 9 | Практикум по ДП и графам | Задача коммивояжера, метод критического пути (CPM) |

### Раздел 2: Задачи в условиях неопределенности (Лекции 10-17)

| № | Тема лекции | Ключевые понятия |
|---|-------------|------------------|

| 12 | Методы решения матричных игр | Седловая точка, смешанные стратегии, графический метод |
| 13 | Теория принятия решений | Критерии Вальда, Гурвица, Сэвиджа, EMV, EOL |
| 14 | Практикум по СМО | Модели M/M/1, M/M/m, формулы Литтла, экономический анализ |

## Особенности курса

### Теоретическая база
Каждая лекция содержит:
- Подробное изложение теоретического материала
- Математические определения и формулы
- Классификации и типологии методов
- Условия применимости различных подходов

### Практическая направленность
Курс включает:
- **Разобранные примеры** с пошаговым решением типовых задач
- **Задачи для самостоятельной работы** после каждой лекции
- Практические рекомендации по применению методов
- Примеры из реальной жизни (бизнес, логистика, производство)

### Междисциплинарный подход
Материал курса связан с:
- Экономикой (оптимизация производства, ценообразование)
- Менеджментом (принятие решений, управление проектами)
- Информатикой (алгоритмы, численные методы)
- Статистикой (прогнозирование, анализ данных)

## Требования к студентам

### Предварительные знания
- Математический анализ (производные, экстремумы функций)
- Линейная алгебра (матрицы, системы линейных уравнений)
- Теория вероятностей и математическая статистика (базовый уровень)
- Основы работы с электронными таблицами (Excel)

### Компетенции, формируемые курсом
После успешного освоения курса студент будет способен:
- Формализовать практические задачи в виде математических моделей
- Выбирать подходящие методы оптимизации для различных типов задач
- Применять алгоритмы теории графов для решения сетевых задач
- Анализировать системы массового обслуживания и рассчитывать их характеристики
- Строить прогнозы на основе временных рядов
- Принимать обоснованные решения в условиях неопределенности и риска
- Интерпретировать результаты моделирования для принятия управленческих решений

## Методические рекомендации

### Для студентов

1. **Последовательное изучение:** Материал курса выстроен по принципу "от простого к сложному". Рекомендуется изучать лекции последовательно, не пропуская разделы.

2. **Решение задач:** После каждой лекции обязательно решайте задачи для самостоятельной работы. Это ключ к пониманию методов.

3. **Практическая работа:** Старайтесь реализовывать алгоритмы в Excel или на языках программирования (Python, MATLAB). Это углубит понимание.

4. **Связь с реальностью:** При изучении каждого метода задавайте себе вопрос: "Где это можно применить на практике?"

5. **Конспектирование:** Выписывайте ключевые формулы и алгоритмы в отдельную тетрадь для быстрого доступа.

### Для преподавателей

1. **Адаптация материала:** Курс можно адаптировать под конкретную специальность, добавляя примеры из соответствующей предметной области.

2. **Лабораторные работы:** Рекомендуется дополнить лекции лабораторными работами с использованием Excel Solver, Python (библиотеки scipy, pulp) или специализированных пакетов (MATLAB, R).

3. **Проектная работа:** В качестве итоговой аттестации можно предложить студентам разработать математическую модель для решения реальной задачи из их будущей профессиональной деятельности.

4. **Визуализация:** Используйте графики, диаграммы и анимации для визуализации алгоритмов (особенно для симплекс-метода, алгоритмов на графах).

5. **Кейс-метод:** Дополните теоретический материал разбором реальных бизнес-кейсов.
